# 🖥️ 定时屏幕截图工具

这是一个基于C++和Qt框架开发的定时屏幕截图工具。它允许用户设置定时截图间隔，选择截图显示器，自定义保存目录，并支持手动截图。程序可以运行在Windows、Linux和macOS系统上。

## 主要功能

- **定时自动截图**：可设置1分钟到24小时的截图间隔，自动保存当前屏幕为图片。

- **手动截图**：随时点击按钮手动保存当前屏幕。

- **自定义保存目录**：支持选择自定义的截图保存文件夹。

- **多显示器支持**：支持选择截屏的显示器和桌面，可以截取单个显示器或所有显示器。

- **系统托盘运行**：程序可以最小化到系统托盘，在后台运行。

- **截图通知**：可选截图成功时的系统通知。

- **截图计数**：显示已截图的次数。

- **启动时立即截图**：可选在开始定时截图时立即截取一张。

- **复制到剪贴板**：可选截图时同时复制到剪贴板。

## 快速使用

从 Releases 页面下载适合您系统的版本

### 基本使用
1.启动程序: 双击运行 ScreenCaptureTool

2.设置间隔: 在"截图间隔"中输入分钟数（1-1440）

3.选择显示器: 从下拉列表中选择目标显示器（0=所有显示器）

4.设置目录: 点击"浏览"选择截图保存位置

5.开始截图: 点击"开始定时截图"按钮

### 系统托盘
- 双击图标: 显示/隐藏主窗口

- 右键菜单: 快速操作菜单

- 隐藏窗口: 程序可在后台运行

## 截图默认保存位置
### windows 系统
C:\Users\你的用户名\Desktop\屏幕截图\
### macOS 系统
/Users/你的用户名/Desktop/屏幕截图/
### Linux 系统
/home/你的用户名/Desktop/屏幕截图/

## 配置文件
程序设置保存在以下位置：

Windows: %APPDATA%\ScreenCaptureTool\Settings.ini

macOS: ~/Library/Preferences/ScreenCaptureTool/Settings.plist

Linux: ~/.config/ScreenCaptureTool/Settings.conf 

## 常见问题
### Q: 截图功能无效？
A: 请检查：
- 是否有屏幕录制权限（macOS需要在系统偏好设置中授权）
- 防病毒软件是否阻止了截图功能
- 是否在多显示器环境下选择了正确的显示器

### Q: 程序无法启动？
A: 请确保：
- 已安装必要的运行库（Windows可能需要VC++ Redistributable）
- 有足够的磁盘空间保存截图
- 对目标目录有写入权限

### Q: 如何更改截图格式？
A: 目前默认使用PNG格式以保证质量。如需其他格式，可通过修改源代码并重新编译实现。

### Q: 支持快捷键吗？
A: 当前版本支持系统托盘菜单操作。可通过修改源代码添加全局快捷键支持。
